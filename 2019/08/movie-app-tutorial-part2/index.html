  1. MaterialApp: Widget Treeの最上位。テーマやルーティング定義

  1. Scaffold: Materialデザインの画面レイアウトを定義するWidget 基本はappBarとbodyが含まれます。 また、 Drawer、 Snackbar, BottomSheetを使用する為、APIも用意されています。

ホーム画面レイアウト

class HomePage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Popular'),
      ),
      // 1.
      body: Column(
        crossAxisAlignment: CrossAxisAlignment.stretch,
        children: <Widget>[
          // 2.
          Container(height: 40, color: Colors.blue, child: Center(child: Text('ショートカットボタン'))),
          // 3.
          Expanded(child: Container(color: Colors.green, child: Center(child: Text('Movie一覧')))),
        ],
      ),
    );
  }
}
  1. Column:子供を縦方向並んで、反対はRowWidgetがあります。
  2. 定義の通り、高さ40の青ボックス(Container)を用意し、真ん中(Center)にテキスト(Text)を配置します。
  3. Expanded:Columnは高さ変動するので、可能な範囲、全スペースを使用したい時使用されるWidgetです。

  1. ListView:子供を並んで表示するが、大量データを持って、スクロールができる特徴です。
